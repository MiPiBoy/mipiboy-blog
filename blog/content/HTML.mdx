# اچ تی ام ال (HTML)

HTML (زبان نشانگذاری متنی فراترند) بنیادی‌ترین زبان برای خلق صفحات وب است. تگ های HTML به صفحه اعطا می‌کنند و به مرورگر نشان می‌دهند که از این المان چیستی در صفحه.

---

> برای مشاهده **لیست کامل تمام تگ‌ها**، می‌توانید به منبع **MDN Web Docs** مراجعه کنید:  
> [https://developer.mozilla.org/en-US/docs/Web/HTML/Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element)

## تگ های اصلی و مهم

### تگ های اساسی
1. `<!DOCTYPE html>` - نعت نسخه استاندارد HTML
2. `<html>` - ریشه سند وب
3. `<head>` - شامل متاداتا سند (مثل تیتر و لینک به CSS)
4. `<body>` - حاوی محتوای اصلی صفحه

### تگ های تصویری متن
5. `<h1>` تا `<h6>` - عناوین از بزرگ تا کوچک
6. `<p>` - پاراگراف
7. `<div>` - بخش بندی عمومی
8. `<span>` - بخش بندی درون متنی
9. `<br>` - شکستن خط

### تگ های لینک و تصاویر
10. `<a>` - ایجاد لینک
11. `<img>` - اضافه کردن تصویر

### تگ های فرم
12. `<form>` - ایجاد فرم
13. `<input>` - فیلد ورودی
14. `<textarea>` - فیلد متن بزرگ
15. `<button>` - دکمه
16. `<select>` - لیست بازشویی
17. `<option>` - گزینه در لیست بازشویی

### تگ های جدول
18. `<table>` - جدول
19. `<tr>` - سطر جدول
20. `<td>` - سلول جدول
21. `<th>` - سرستون جدول

### تگ های لیست
22. `<ul>` - لیست غیر مرتب
23. `<ol>` - لیست مرتب
24. `<li>` - آیتم لیست

### تگ های تعریف سمانت
25. `<header>` - سرآینجا
26. `<nav>` - منوی صفحه
27. `<main>` - محتوای اصلی
28. `<article>` - مقاله و محتوای مستقل
29. `<section>` - بخش مجزای صفحه
30. `<aside>` - محتوای جانبی
31. `<footer>` - پانویستی

### تگ های متا
32. `<meta>` - متاداتا سند
33. `<link>` - ارتباط به فایل خارجی (مثل CSS)
34. `<script>` - اضافه کردن JavaScript
35. `<style>` - اضافه کردن CSS درون سند

### تگ‌های چندرسانه‌ای
36. `<video>` - برای نمایش فیلم
37. `<audio>` - برای پخش صوت

### تگ‌های متا و اطلاعاتی
38. `<title>` - عنوان صفحه در تب مرورگر
39. `<base>` - پایه URL برای همه لینک‌های صفحه

### تگ‌های نمایش ریاضی
40. `<svg>` - برای ایجاد گرافیک‌های وکتوری
41. `<canvas>` - برای ترسیم گرافیک‌های پویا

### تگ‌های پیشرفته
42. `<template>` - برای ساخت قطعات کد قابل استفاده مجدد
43. `<datalist>` - برای ایجاد لیست پیشنهادی در فیلدها

### تگ‌های معنایی جدید (HTML5)
44. `<mark>` - برای هایلایت کردن متن
45. `<progress>` - نمایش نوار پیشرفت
46. `<meter>` - نمایش مقدار در یک محدوده
47. `<details>` - ایجاد قسمت جمع شونده
48. `<summary>` - عنوان قسمت جمع شونده

### تگ‌های فرم پیشرفته
49. `<output>` - نمایش نتیجه محاسبات
50. `<fieldset>` - گروه بندی فیلدهای فرم
51. `<legend>` - عنوان گروه فیلدها

### تگ‌های ناوبری
52. `<map>` - ایجاد نقشه تصویر
53. `<area>` - ایجاد ناحیه فعال در نقشه تصویر
